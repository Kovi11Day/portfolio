Location Mention Recognition with BERT

Project Description

This project involves developing a Location Mention Recognition system using BERT for detecting toponyms (place/ area/ street names) in microblogging posts. The aim is to help authorities determine specific locations to send resources such as medical aid, food.

Handwriting Recognition using Hidden Markov Model

Project Description

Given 3 HMMs representing the 0, 1 and 7 digits and sequences of observations, our goal is to classify these observations according to the digits they represent: 0,1 or 7. The challenge of this project is coding the Viterbi algorithm from scratch.
